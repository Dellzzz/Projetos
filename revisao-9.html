Equações do 2º grau — incompletas

Quando falta o b ou o c

Toda equação do 2º grau tem a forma ax² + bx + c = 0, com a ≠ 0. Ela é incompleta quando o coeficiente b ou o c (ou os dois) é igual a zero. Nesses casos, dá para resolver de um jeito mais rápido, sem precisar de Bhaskara.

TipoFormaComo resolver
c = 0ax² + bx = 0fator comum: x · (ax + b) = 0
b = 0ax² + c = 0isolar o x²: x² = −c/a
b = 0 e c = 0ax² = 0x = 0 (raiz dupla)

Tipo 1 · falta o c (ax² + bx = 0)

Coloque o x em evidência (fator comum). Um produto é zero quando um dos fatores é zero — então uma das raízes é sempre x = 0.

Exemplo
2x² − 8x = 0
x · (2x − 8) = 0
x = 0   ou   2x − 8 = 0 → x = 4
Raízes: x = 0 e x = 4

Tipo 2 · falta o b (ax² + c = 0)

Deixe o x² sozinho de um lado e tire a raiz quadrada (lembrando do ±). Só existe raiz real se o número for positivo.

Tem solução
x² − 9 = 0 → x² = 9
x = ±√9
Raízes: x = −3 e x = 3
Sem solução real
x² + 4 = 0 → x² = −4
não existe √ de número negativo
Sem raízes reais
Resumo: na incompleta com c = 0, sempre aparece o x = 0. Na incompleta com b = 0, as raízes são simétricas (uma positiva e uma negativa, como −3 e 3) — ou não existem, se o x² ficar igual a um número negativo.

Equações completas — fórmula de Bhaskara

O método algébrico para ax² + bx + c = 0

Quando a equação é completa (tem a, b e c diferentes de zero), usamos a fórmula de Bhaskara. Primeiro calculamos o discriminante Δ (lê-se "delta") e depois aplicamos a fórmula:

Δ = b² − 4ac
x = −b ± √Δ2a

O sinal ± quer dizer que fazemos a conta duas vezes: uma somando √Δ e outra subtraindo. O valor de Δ já avisa quantas raízes a equação tem:

Δ > 0

duas raízes reais diferentes

Δ = 0

uma raiz real (dupla)

Δ < 0

nenhuma raiz real

Resolva por Bhaskara teste

Escolha os coeficientes de ax² + bx + c = 0 e acompanhe a conta:

Δ > 0 · duas raízes
x₁ = 3    x₂ = 2

Exemplos resolvidos

Duas raízes (Δ > 0)
2x² + 5x − 3 = 0  (a=2, b=5, c=−3)
Δ = 5² − 4·2·(−3) = 25 + 24 = 49
x = (−5 ± √49) / 4 = (−5 ± 7) / 4
x₁ = 1/2   e   x₂ = −3
Uma raiz dupla (Δ = 0)
x² − 6x + 9 = 0  (a=1, b=−6, c=9)
Δ = (−6)² − 4·1·9 = 36 − 36 = 0
x = (6 ± 0) / 2
x = 3 (raiz dupla)

Triângulos retângulos

O triângulo com um ângulo de 90°

Um triângulo retângulo é aquele que tem um ângulo reto (90°). Os lados têm nomes especiais:

• A hipotenusa é o lado oposto ao ângulo reto — e é sempre o lado maior.

• Os catetos são os dois lados que formam o ângulo reto.

Como a soma dos ângulos de qualquer triângulo é 180°, os dois ângulos que não são retos são agudos e, juntos, somam 90°. Mexa nos catetos abaixo e veja tudo mudar:

8cateto c

Teorema de Pitágoras

A relação entre os lados do triângulo retângulo

Em todo triângulo retângulo, o quadrado da hipotenusa é igual à soma dos quadrados dos catetos:

a² = b² + c² a = hipotenusa  ·  b e c = catetos

Veja com o famoso triângulo 3, 4, 5: o quadrado de cada cateto, somado, dá exatamente o quadrado da hipotenusa. Conte os quadradinhos!

5² = 25hipotenusa
9 + 16 = 25 ✔ Confere! Trios de números inteiros que fecham certinho assim são chamados de ternos pitagóricos: (3, 4, 5), (5, 12, 13), (8, 15, 17) — e seus múltiplos, como (6, 8, 10).

Achar a hipotenusa teste

Você tem os dois catetos e quer o lado maior: a = √(b² + c²).

a = 5

Achar um cateto teste

Você tem a hipotenusa e um cateto e quer o outro: c = √(a² − b²).

c = 12

Exemplos do dia a dia

A escada na parede
Escada de 5 m, com a base a 3 m da parede. Que altura ela alcança?
altura = √(5² − 3²) = √(25 − 9) = √16
Altura = 4 m
O portão
Um portão de 2 m por 1,5 m. Qual o tamanho da travessa na diagonal?
d = √(2² + 1,5²) = √(4 + 2,25) = √6,25
Diagonal = 2,5 m
A distância reta
Ando 8 km para o leste e 6 km para o norte. Quão longe estou do início?
d = √(8² + 6²) = √(64 + 36) = √100
Distância = 10 km
